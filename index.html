VR game for Meta Quest 2 & 3

Infinity Game

Infinity Game is a stylized VR climbing game where players climb an endless alien structure using hand-based locomotion, avoid traps, collect rewards and try to reach the highest point possible.

About the developer

Leaf and Light Studio is a solo independent studio focused on stylized, high-quality VR experiences for Meta Quest.

Contact: leafandlightstudio@gmail.com

Platform & features

  • Platform: Meta Quest 2 & Meta Quest 3
  • Mode: Single-player, offline gameplay with optional online services
  • Online services:
    • Meta in-app purchases (add-ons, e.g. “Golden Key”)
    • Meta cloud saves (if enabled)
    • Meta leaderboards (online rankings, planned)
